##########################    使用方法   #######################

1.clone or download 下载到本地
2. 将主题解压后 拷贝 到  C:\Users\Administrator\.vscode\extensions  目录下
3. 打开 VSCode 使用[ctrl + shift + p] 快捷键 输入 "reload window" 回车  (此时VSCode 会重新加载一次)
4. 使用 [ctrl+K ctrl+t] 组合键 此时会弹出 主题对话框
5. 向下 选择 Wilson 代码高亮改变 主题设置成功

****   注: 解压后 最外层文件夹为 WilsonDarkTheme-master   ****
