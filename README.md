##########################    使用方法   #######################

1. 将主题解压后 拷贝 到  C:\Users\Administrator\.vscode\extensions  目录下
   1.1 此时最外层文件夹为 WilsonDarkTheme-master
2. 打开 VSCode 使用 [ctrl+K ctrl+t] 组合键 此时会弹出 主题对话框
3. 向下 选择 Wilson 
4. 代码变样了   哈哈 ^_^
5. 祝您使用愉快 !
